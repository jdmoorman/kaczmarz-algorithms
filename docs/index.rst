Installation
------------

To install Kaczmarz Algorithms, run this command in your terminal:

.. code-block:: console

    $ pip install kaczmarz-algorithms

This is the preferred method to install Kaczmarz Algorithms, as it will always install the most recent stable release.

If you don't have `pip`_ installed, these `installation instructions`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _installation instructions: http://docs.python-guide.org/en/latest/starting/installation/
